> #### 
> 
> Two classes $x$ and $y$ are identical if they consists of exactly the same elements. In symbols,
> 
> $$
> \forall x\; \forall y\; \left(\underline{\forall z \; (z \in x \iff x \in y)} \implies x = y \right).
> $$
